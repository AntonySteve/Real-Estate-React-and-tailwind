## вҡҷпёҸ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/antonysteve/Real-Estate-React-and-tailwind.git
```

Navigate to the project folder:

```bash
cd Real-Estate-React-and-tailwind
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
